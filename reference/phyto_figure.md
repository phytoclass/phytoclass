# Phytoplankton Class Abundance Figure

This function plots the class abundances as output by
`simulated_annealing`.

## Usage

``` r
phyto_figure(c_matrix)
```

## Arguments

- c_matrix:

  C matrix of class abundance concentrations

## Value

A stacked line plot with sample number on x axis, chl a concentrations
on y axis, and phytoplankton groups as colors
